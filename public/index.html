üìö API Reference

        

          
          

            

              GET
              /tickers
            

            
Retrieve historical price data for a specific ticker symbol.

            

              

                Parameters:
                

                  
symbol - Ticker symbol (required)

                  
interval - Time interval: 1D, 1H, 1m (required)

                  
start_date - Start date YYYY-MM-DD (optional)

                  
format - Response format: json, csv (default: json)

                

              

            

          


          
          

            

              GET
              /tickers/group
            

            
List all available ticker symbols grouped by data availability.

            

              

                Returns:
                Groups of tickers with their available intervals
              

            

          


          
          

            

              GET
              /health
            

            
Check API health status and cache performance metrics.

            

              

                Returns:
                System status, cache stats, and data availability
              

            

          


          
          

            

              GET
              /raw/*
            

            
Legacy GitHub proxy for backward compatibility.

            

              

                Note:
                Deprecated - use /tickers endpoint instead
