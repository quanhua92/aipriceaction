üìö API Reference

        

          
          

            

              GET
              /tickers
            

            
Retrieve historical price data with technical indicators.

            

              

                Parameters:
                

                  
symbol - Ticker symbol (optional, defaults to all tickers)

                  
interval - Time interval (optional, default: 1D)

                  
Base: 1D, 1H, 1m

                  
Minute aggregations: 5m, 15m, 30m

                  
Day aggregations: 1W, 2W, 1M

                  
start_date - Start date YYYY-MM-DD (optional)

                  
end_date - End date YYYY-MM-DD (optional, defaults to today)

                  
limit - Number of records to return (optional, ignored if start_date provided)

                  
format - Response format: json, csv (default: json)

                  
legacy - Legacy price format (divides by 1000) (optional, default: false)

                  
cache - Use cache (optional, default: true)

                

              

            

          


          
          

            

              GET
              /tickers/group
            

            
List all available ticker symbols grouped by data availability.

            

              

                Returns:
                Groups of tickers with their available intervals
              

            

          


          
          

            

              GET
              /health
            

            
Check API health status and cache performance metrics.

            

              

                Returns:
                System status, cache stats, and data availability
              

            

          


          
          

            

              GET
              /analysis/top-performers
            

            
Get top/bottom performing stocks with customizable sorting.

            

              

                Parameters:
                

                  
date - Analysis date YYYY-MM-DD (optional, defaults to latest)

                  
sort_by - Metric to sort by (optional, default: close_changed)

                  
Options: close_changed, volume, volume_changed, ma10_score, ma20_score, ma50_score, ma100_score, ma200_score

                  
direction - Sort direction: asc, desc (optional, default: desc)

                  
limit - Number of results (optional, default: 10, max: 100)

                  
sector - Filter by sector name (optional)

                  
min_volume - Minimum trading volume (optional, default: 10000)

                

              

            

          


          
          

            

              GET
              /analysis/ma-scores-by-sector
            

            
Moving average analysis grouped by stock sectors.

            

              

                Parameters:
                

                  
date - Analysis date YYYY-MM-DD (optional, defaults to latest)

                  
ma_period - MA period: 10, 20, 50, 100, 200 (optional, default: 20)

                  
min_score - Minimum MA score threshold (optional, default: 0.0)

                  
above_threshold_only - Only stocks above threshold (optional, default: false)

                  
top_per_sector - Max stocks per sector (optional, default: 10, max: 50)
